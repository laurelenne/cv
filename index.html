A propos de moi

Bienvenue sur mon site !

Depuis toujours passionnée par l'informatique et les nouvelles technologies, j'ai décidé de me réaliser dans ce domaine afin de m'épanouir dans mon futur métier. Suite à un cursus dans le secteur social, j'ai entamé une reconversion dans le développement. Cette reconversion est l'occasion pour moi de mêler passion et projet professionnel.
Tout d'abord, j'ai suivi une formation "Projet Pro Numérique", qui m'a permis de structurer mon projet professionnel et de commencer mon apprentissage du code avec un formateur.

J'ai ensuite intégré la formation "Développeur web et mobile" du centre de formation ADRAR à Montpellier dans le but de me professionnaliser dans le développement web.
Aujourd'hui, je suis à la recherche de nouvelles opportunités professionnelles dans le développement web.
